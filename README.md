# 🌍 Travel Buddy — BCA Final Project

Travel Buddy is a web-based travel booking application developed as a Final Year BCA Project. It allows users to explore travel destinations, view packages, and book trips online.

---

## ✨ Features

- 🔐 **User Login & Registration** — Secure sign up and login for users
- 📦 **View Packages** — Browse available travel packages
- 🗺️ **View Destinations** — Explore popular travel destinations with images
- 📅 **Book Travel Packages** — Easy online booking system
- 🛠️ **Admin Dashboard** — Manage destinations, packages and bookings
- 📬 **Contact Form** — Users can reach out with queries

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| HTML & CSS | Frontend design and layout |
| PHP | Backend logic and server-side scripting |
| MySQL | Database for storing users, bookings, packages |
| XAMPP | Local server environment |

## 🚀 How to Run This Project

1. Download and install **XAMPP** from [apachefriends.org](https://www.apachefriends.org)
2. Clone or download this repository
3. Copy the project folder to `C:/xampp/htdocs/`
4. Start **Apache** and **MySQL** from XAMPP Control Panel
5. Import the database:
   - Open `http://localhost/phpmyadmin`
   - Create a new database
   - Import the `.sql` file from the project folder
6. Open your browser and go to:
   ```
   http://localhost/tourism_project/
   ```
